Intro

Python is a general purpose language. Python is a dynamically typed interpreted language. It was created by Guido van Rossum in the year 1991. Python is famous for its applications in machine learning and data science world, and in the scientific study for research, stimulations, etc. Web based applications like youtube, instagram and dropbox also use python in the backend. Python has simple easy to read syntax. Which makes it a very good first programming language to learn.
