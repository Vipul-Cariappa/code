loops

In some cases doing a task repeatedly is required. Example to get sum of list of numbers. For doing tasks which are repetitive in nature python gives loops. There are two types of while loops and for loops. The while loop in Python is used to iterate over a block of code as long as the test expression (condition) is true. 


# summing n natural numbers using while loop
n = int(input(“Enter a number: “))

sum = 0

while n > 0:
    sum += n
    n -= 1

print(sum)
The for loop in Python is used to iterate over a sequence (list, tuple, string) or other iterable objects. 

# summing a list of numbers
x = [12, 34, 4, 57, 78, 45, 65]

sum = 0

for i in x:
    sum += i

print(sum)
Another example: 

# summing n natural numbers using for loop

n = int(input(“Enter a number: “))

sum = 0

for i in range(n):
    sum += i

print(sum)
