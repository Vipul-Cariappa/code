Variables And Operations

Python offers a wide range of variable types for storing. 


integer = 3 # int
number = 3.3 # float
string = “Hello World!” # str
complex = 4j + 2 # complex number
truth = True # boolean
Python also offers data structures. 

collection = [1, 2, 3, “hello”, 3.3] # list
Immutable = (1, 2, 3, “hello”, 3.3) # tuple
dictionary = {“hello”: “world”, “program”: “python”} # dictionary
Operations on numbers 

x = 27
y = 4
            
print(x + y) # Addition
print(x - y) # subtraction
print(x * y) # multiplication
print(x / y) # division
print(x % y) # reminder / modulus
print(x // y) # division integer
print(x ** y) # exponent
Note: You can concatenate strings by 

first_name = “Donald ”
last_name = “Trump”
print(first_name + last_name)
